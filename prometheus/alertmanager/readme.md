# AlertManager:

Alertmanager is a core component in the Prometheus monitoring ecosystem, responsible for handling alerts generated by Prometheus server. 

Alertmanager is a service that,

* Receives alerts from Prometheus.

* Deduplicates, groups, and routes them.

* Sends notifications to external services (email, Slack, PagerDuty, etc.).

* Manages silencing, inhibition, and alert escalation.
